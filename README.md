# Odama Dashboard Login Page

A modern, responsive login page for the Odama Team Shot Portal built with React and Tailwind CSS.

## Features

- Clean, minimalist design based on Figma mockup
- Responsive layout that works on all screen sizes
- Interactive form with username and password inputs
- Smooth animations and transitions
- Built with React 18 and Tailwind CSS 3
- Uses Vite for fast development and optimized builds

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

This will start the development server at `http://localhost:5173`

## Build

```bash
npm run build
```

## Project Structure

```
src/
├── App.jsx                 # Main app component
├── main.jsx               # Entry point
├── index.css              # Global styles and Tailwind directives
└── components/
    └── LoginPage.jsx      # Login page component
```

## Configuration Files

- `vite.config.js` - Vite configuration
- `tailwind.config.js` - Tailwind CSS configuration
- `postcss.config.js` - PostCSS configuration
- `package.json` - Project dependencies and scripts
- `index.html` - HTML template

## Design

The login page features:
- Odama® logo at the top
- Username input field (rounded style)
- Password input field (rounded style)
- Login button with arrow icon
- Footer text: "Odama® Team Shot Portal"
- Light background (#f9f9f9) with smooth interactions
